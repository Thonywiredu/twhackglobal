# TWHack Cyber Defense Website (Next.js + Tailwind)

## Run locally
1. Install Node.js (LTS).
2. In this folder:
   - npm install
   - npm run dev
3. Open http://localhost:3000

## Deploy (Vercel)
1. Create a GitHub repo and push this project.
2. Sign into Vercel with GitHub.
3. Import the repo → Deploy.
4. Optional: add a custom domain in Vercel settings.
